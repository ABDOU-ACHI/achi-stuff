Nmap (Network Mapper)

Network Discovery & Security Auditing
Intermediate

Nmap is a free and open-source network discovery and security auditing tool. It's used to discover hosts and services on a computer network by sending packets and analyzing responses.

Key Features

  • Host discovery and port scanning
  • Service and version detection
  • Operating system detection
  • Scriptable with NSE (Nmap Scripting Engine)
  • Firewall/IDS evasion techniques

Essential Commands

nmap -sS target.com SYN stealth scan
nmap -sV -sC target.com Version detection with default scripts
nmap -A target.com Aggressive scan (OS detection, version, script, traceroute)
nmap --script vuln target.com Run vulnerability detection scripts

Common Use Cases

Network Discovery

Identify live hosts and open ports on target networks

Service Enumeration

Detect running services and their versions

Vulnerability Scanning

Identify potential security vulnerabilities

Metasploit Framework

Penetration Testing Platform
Advanced

Metasploit is the world's most used penetration testing framework. It's a powerful tool for developing and executing exploit code against remote targets.

Key Features

  • Extensive exploit database
  • Payload generation and encoding
  • Post-exploitation modules
  • Database integration for tracking
  • Web interface (Metasploit Pro)

Essential Commands

msfconsole Start Metasploit console
search type:exploit platform:windows Search for Windows exploits
use exploit/windows/smb/ms17_010_eternalblue Use specific exploit module
set PAYLOAD windows/meterpreter/reverse_tcp Set payload for exploit

Typical Workflow

1

Reconnaissance

Gather information about the target

2

Vulnerability Assessment

Identify exploitable vulnerabilities

3

Exploitation

Execute exploit against target

4

Post-Exploitation

Maintain access and gather intelligence

Burp Suite

Web Application Security Testing
Intermediate

Burp Suite is an integrated platform for performing security testing of web applications. It contains various tools that work together to support the entire testing process.

Key Components

  • Proxy - Intercept and modify HTTP/HTTPS traffic
  • Spider - Automated crawling of web applications
  • Scanner - Automated vulnerability scanner (Pro version)
  • Intruder - Customizable attack tool
  • Repeater - Manual request manipulation
  • Sequencer - Token randomness analysis

Core Modules

Proxy

Intercept and modify web traffic between browser and server

Spider

Automatically crawl web applications to map content

Intruder

Perform automated attacks against web applications

Repeater

Manually manipulate and resend HTTP requests

Common Attack Types

SQL Injection Testing

Test for SQL injection vulnerabilities in web forms and parameters

Cross-Site Scripting (XSS)

Identify reflected, stored, and DOM-based XSS vulnerabilities

Authentication Bypass

Test for weak authentication mechanisms and session handling

Installation & Setup Guide

Kali Linux Setup

Most tools come pre-installed with Kali Linux. Update your system regularly:

sudo apt update && sudo apt upgrade -y
sudo apt install -y metasploit-framework nmap wireshark burpsuite

Ubuntu/Debian Installation

Install tools individually on Ubuntu/Debian systems:

# Install Nmap
sudo apt install nmap

# Install Metasploit
curl https://raw.githubusercontent.com/rapid7/metasploit-omnibus/master/config/templates/metasploit-framework-wrappers/msfupdate.erb > msfinstall && chmod 755 msfinstall && ./msfinstall

Windows Installation

Some tools have Windows versions available:

# Download Nmap for Windows from: https://nmap.org/download.html
# Download Burp Suite Community from: https://portswigger.net/burp
# Consider using WSL2 with Kali Linux for better compatibility

macOS Installation

Use Homebrew to install security tools on macOS:

# Install Homebrew first: https://brew.sh/
brew install nmap wireshark
brew install --cask burp-suite
