Red Hat Certification Path

Parallel to main curriculum

RHCSA (Red Hat Certified System Administrator)

Foundation certification covering essential system administration skills.

  • System configuration and management
  • File systems and storage management
  • User and group management
  • Network services configuration
Foundation Level EX200

RHCE (Red Hat Certified Engineer)

Advanced certification focusing on automation and enterprise services.

  • Ansible automation
  • Advanced networking
  • Security hardening
  • Performance tuning
Advanced Level EX294

Red Hat Certified Specialist in Identity Management

Specialized certification in identity and access management.

  • IdM server installation and configuration
  • User and host management
  • Certificate management
  • Integration with Active Directory
Specialist Level EX342
